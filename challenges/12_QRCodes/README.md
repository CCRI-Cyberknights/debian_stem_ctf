# 🧩 Challenge 12: QR Code Puzzle

**Mission Briefing:**
You have recovered a set of QR (Quick Response) codes. QR codes are just a visual way of encoding text data. One of these images contains the agency flag; the others are decoys or broken links.

## 🧠 Intelligence Report
* **The Lock:** The data is encoded visually in 2D barcodes.
* **The Strategy:** **Bulk Scanning**. You have multiple images, and checking them one by one is inefficient.
* **The Warning:** This folder contains **decoy QR codes** that yield fake flags. You must verify which one is the real deal.

**Your Goal:** Scan the images, filter the output, and find the one that decodes to a valid flag.

## 📂 Files in This Folder
* `qr_01.png` – `qr_05.png`: Image files containing QR codes.

---

**🏁 Flag Format:** `CCRI-AAAA-1111`
